Docker 常用命令和参数

2018-08-06

记录下在使用过程中用到的命令和参数用法。

常用命令

  1. 搜索镜像
    docker search *
  2. 拉取镜像
    docker pull *
  3. 拉取指定标签镜像
    docker pull *:tag
  4. 查询本地镜像列表
  5. 运行镜像（配合参数使用）
    docker run *:tag
  6. 运行镜像并进入命令行
    docker run -it *:tag
  7. 暂存并退出容器
    ctrl+p ctrl+q
    （容器暂存后在后端运行并为退出）
  8. 查询容器列表
    docker ps 查询运行中的容器
    docker ps -a 查询所有容器
  9. 再次进入暂存的容器
    docker attach [CONTAINER ID]
  10. 退出容器
    直接在容器中运行exit即可退出容器
  11. 重启容器
  12. 停止容器
    docker stop [CONTAINER ID]
  13. 进入一个在运行中的容器的命令行
    docker exec -it [CONTAINER ID] bash
  14. 删除容器
    docker rm [CONTAINER ID]
  15. 删除镜像
    docker rmi [IMAGE ID]
  16. 保存修改后的容器为镜像
    docker commit [CONTAINER ID] [NEW REPOSITORY]
