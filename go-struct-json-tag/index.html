Go struct 中 json 标记解释

2019-01-30
  • 解析（encode/decode） 的时候，使用 “myName”，而不是 “Field”
  • 解析（encode/decode） 的时候，使用 “myName”，而不是 “Field”
  • 并且如果字段值是空的话则忽略
  • 解析（encode/decode） 的时候，使用 “Field”
  • 但是字段值是空的话则忽略
  • 注意逗号.
  • 忽略字段
  • 解析（encode/decode） 的时候，使用 “-“，而不是 “Field”
  • json中是string类型,struct中是int类型
