Go学习杂记

2018-12-14
  1. 方法
    方法实际上也是函数,只是在声明时,关键字func喝方法名之间增加了一个参数(接收者)
  2. defer 关键字
    在函数或方法中,用defer标识的语句将会在函数或方法返回后执行
  3. chan 通道
  4. select 语句
    select 中没有default分支,程序将会阻塞等待 select 中有default分支,程序将会将阻塞转变为非阻塞,如果通道有内容则接收,没有则执行default分支
